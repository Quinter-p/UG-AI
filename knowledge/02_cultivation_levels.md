# 修炼境界体系

## 凡界 / 人界常见境界
| 境界 | 知识库解释 | AI 回答时的判断 |
|---|---|---|
| 炼气期 | 修仙入门阶段，法力有限，依赖基础法术、符箓、低阶法器和宗门资源。 | 不建议主动卷入高阶冲突，重点是保命、积累、找功法和丹药。 |
| 筑基期 | 真正踏入修仙大道的重要门槛，法力、神识、寿元和行动能力明显提升。 | 可以承担低中阶任务，但仍要避开结丹以上修士。 |
| 结丹期 | 中高阶修士，通常开始拥有更稳定的地位和较强法宝能力。 | 可以成为小宗门长老级战力，但仍受元婴压制。 |
| 元婴期 | 高阶修士，在人界通常属于顶级或接近顶级存在。 | 具备强大威慑力，低阶修士应避免直接冲突。 |
| 化神期 | 人界极高层次，与飞升、空间节点、更高世界联系紧密。 | 属于战略级存在，普通修士不应主动接触。 |

## 灵界常见境界
| 境界 | 知识库解释 | AI 回答时的判断 |
|---|---|---|
| 炼虚期 | 进入更高层级修炼体系后的重要阶段。 | 已属强者，但在灵界仍需依附大势力或谨慎行动。 |
| 合体期 | 灵界高阶强者。 | 可影响一方局势，但仍需防范大乘存在。 |
| 大乘期 | 灵界顶尖层级，接近飞升门槛。 | 任何低阶修士都应视为不可招惹对象。 |
| 渡劫期 | 大乘圆满后临近飞升的特殊层次。 | 与飞升、天劫、仙界入口相关，极端危险。 |

## 仙界常见境界
| 境界 | 知识库解释 |
|---|---|
| 真仙 | 飞升后真正进入仙人体系的重要阶段。 |
| 金仙 | 更高层级仙人，实力和法则理解提升。 |
| 太乙 | 仙界高阶层级之一。 |
| 大罗 | 极高层级仙人。 |
| 道祖 | 与法则大道高度绑定的顶级存在。 |

## 小境界
除炼气期常以层数划分外，多数境界可分为初期、中期、后期、大圆满。AI 回答时应注意：同一大境界内的小境界差距也可能影响胜负、寿元、神识、法宝操控和任务风险。
